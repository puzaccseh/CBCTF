Module 2: Web Application Penetration Testing ============================================= This module covers the OWASP Top 10 and beyond, with hands-on labs using BurpSuite Pro, OWASP ZAP, and custom tooling developed by the Cyber Bangla team. -------------------------------------------------- Section 2.1 – Cross-Site Scripting (XSS) -------------------------------------------------- Types: Reflected XSS – Server echoes unsanitised user input in the response. Stored XSS – Malicious input is persisted in the database/store. DOM-based XSS – Client-side JavaScript writes attacker input to the DOM.
