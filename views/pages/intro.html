Module 1: Introduction to Ethical Hacking ========================================== Welcome to Cyber Bangla Academy's foundational course on ethical hacking. By the end of this module you will: - Understand the penetration testing lifecycle - Set up a Kali Linux lab environment - Perform basic reconnaissance using OSINT tools - Conduct network scanning with Nmap - Identify common vulnerabilities in web applications Prerequisites: Basic Linux command-line experience. -------------------------------------------------- Section 1.1 – The Penetration Testing Lifecycle -------------------------------------------------- Phase 1: Reconnaissance (recon) Passive recon: Shodan, WHOIS, Google Dorks, LinkedIn Active recon: Nmap, Masscan, Netcat Phase 2: Scanning & Enumeration Port scanning, service detection, OS fingerprinting Web directory bruteforcing (dirb, gobuster, ffuf) Phase 3: Exploitation Manual exploitation vs automated tools Metasploit Framework basics Phase 4: Post-Exploitation Privilege escalation, pivoting, persistence Phase 5: Reporting Writing clear, actionable pentest reports -------------------------------------------------- Section 1.2 – Setting Up Your Lab -------------------------------------------------- Recommended setup: - Kali Linux 2024.x (latest) - VirtualBox or VMware Workstation - Vulnerable VMs: Metasploitable, DVWA, VulnHub Download Kali: https://www.kali.org/get-kali/ Next Module: Web Application Penetration Testing → Access: /academy/content?page=webapp
